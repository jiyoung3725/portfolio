DEV EXPERIENCE
문의, 리뷰 CRUD 기능

사용자가 작성한 문의, 리뷰글은 회원번호와 일치할 때 수정, 삭제가 가능하도록 기능을 구현했습니다.

정렬 기능

최신순, 가격낮은순, 가격높은순, 인기순 중 사용자가 선택한 옵션에 따라 상품이 정렬되도록 구현했습니다.

페이징 기능

한 페이지에 12개의 상품 목록이 보이도록 했고 페이지를 이동해도 사용자가 선택한 옵션이 유지되도록 구현했습니다.

좋아요 기능

사용자가 좋아요버튼을 눌렀을 때 db에 true/false를 저장하고 페이지에 접근할 때 좋아요한 상품 목록이 나타나도록 구현했습니다.

장바구니 기능

장바구니에 추가된 상품을 선택삭제, 품절상품삭제, 전체삭제가 가능하도록 했고 상품을 선택할 때마다 가격이 합산되도록 구현했습니다.

로그인 기능

자체로그인뿐만 아니라, 카카오 로그인 기능을 추가하여 회원의 편의에 맞게 사이트에 접속할 수 있도록 기능을 구현했습니다.
